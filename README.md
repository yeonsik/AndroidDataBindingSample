Android DataBinding Sample
==============

데이터 바인딩 라이브러리 기능에 대한 기본적인 샘플들과 ButterKnife와 비교 샘플들.

개인적으로 공부하는 용도로 만듦. 데이터 바인딩 위주로 작성하여 ButterKinfe 샘플 코드는 미비한 상태임.
